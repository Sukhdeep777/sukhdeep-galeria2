Como lo he hecho

En esta sección, explicaré el proceso que seguí para llevar a cabo este proyecto.

Primero, realicé una investigación exhaustiva sobre el tema y recopilé información relevante. Luego, el siguiente paso fue buscar las imágenes de las cartas TOTS para mi index.html. Una vez colocadas todas las imagenes, verifique si se veia visualmente i correcto, despues me enfoqué a añadir la información de cada jugador, con sucarta i video. Para poder navegar entre las cartas puse unos botones al fondo de cada página para que los usuarios pudieran navegar entre las cartas.Finalmente, puse las páginas de referencias i como lo he hecho

Este proceso me permitió adquirir nuevos conocimientos y habilidades, así como mejorar mi capacidad para trabajar en proyectos similares en el futuro.En resumen, la combinación de investigación, diseño y revisión fue clave para el éxito de este proyecto.
